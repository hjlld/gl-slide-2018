# gl-slide

> webgl slide

## How to Use

- right key <kbd>→</kbd> to next slide
- left key <kbd>←</kbd> to previous slide

## Install Modules

``` bash
$ npm install
```

## Dev

``` bash
$ npm run dev
```

## Build

``` bash
$ npm run build
```
